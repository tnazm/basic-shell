## Usage

```
make all
./shell
```

Then type one of the following commands, followed by an argument if required.

## Commands

### cd
Change directory.

```
cd <directory>
```

### pwd
Print working directory.

```
pwd
```

### exit
Exit the shell.

```
exit
```

### help
Display help.

```
help
```

### mkdir
Create a directory.

```
mkdir <directory>
```

### rmdir
Remove a directory.

```
rmdir <directory>
```

### ls
List files in a directory.

```
ls <directory>
```

### cp
Copy a file.

```
cp <source> <destination>
```

### mv
Move a file.

```
mv <source> <destination>
```

### rm
Remove a file.

```
rm <file>
```
